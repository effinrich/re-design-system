## Import

```js
import { Loader } from '@redesignhealth/ui'
```

## Usage

Loader is a pattern component consisting of the Spinner component and Flex component, and best used as a loading feedback for pages/views or sections.

```jsx
<Loader
  size="md"
  color="primary.500"
  thickness="4px"
  speed="0.65s"
  minH="25vh"
/>
```
